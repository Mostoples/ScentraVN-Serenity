Detail per Target

1. Stress berbasis EDA / Keringat — 77% ✅

Dilatih pada WESAD (Empatica E4, 15 subjek). Hanya pakai fitur EDA + suhu kulit agar cocok dengan smartwatch ScentraVN. Validasi subject-wise: akurasi 77%, F1 0.70. EDA adalah penanda aktivitas saraf simpatik yang kuat untuk stres.

2. Stress berbasis EEG — Heuristik QEEG ⚠️

Menggunakan rasio theta/beta dari Muse (penanda arousal QEEG). Threshold berbasis literatur, belum dilatih sebagai model. Dipakai melengkapi emotion & EDA.

3. Emotion (EEG) — 93% ✅   Mental State (EEG) — 72% ✅   Sleep (EEG) — 60% ✅

Tiga model EEG tervalidasi pada data nyata. Emotion & mental-state dilatih pada hardware Muse yang sama dengan produk; sleep stager pada Sleep-EDF (197 subjek, validasi subject-wise).

4. Cognitive Load (EEG) — 29% lintas-subjek / 61% within ⚠️

Dilatih pada STEW (Emotiv 14ch). Temuan jujur: fitur membawa sinyal workload (61% subject-dependent) TAPI tidak generalisasi antar orang (29%, mendekati chance). Ditandai sebagai advisory.

5. Blood Pressure / Glucose / Hemoglobin — Heuristik / Eksperimental 🔴

Estimasi research-grade dari morfologi PPG. Glucose: R²≈0 pada dataset 22-pasien (jatuh ke heuristik). BP: MAE literatur 8-12 mmHg. Hemoglobin: butuh PPG multi-wavelength (MAX30102 hanya 2λ). Bukan alat medis.

6. Asam Urat — Berbasis input klinis, BUKAN sensor 🔴

Asam urat tidak dapat diukur dari PPG/EDA/sensor optik. Penelitian yang ada memprediksi dari data klinis (usia, BMI, tekanan darah, pola makan). Di app ditampilkan sebagai estimasi risiko berbasis input manual, bukan pembacaan sensor.

Pipeline Training

1
Akuisisi dataset — diunduh via Kaggle API / GitHub / PhysioNet.
2
Ekstraksi fitur — fitur yang IDENTIK dapat dihitung live di browser (band power, HRV, statistik EDA) agar training = inference.
3
Split subject-wise — tidak ada subjek yang sama di train & test (mencegah kebocoran optimistik).
4
Training — scikit-learn MLP / RandomForest. Ceiling diukur dengan fitur penuh, model compact untuk deploy.
5
Export JSON — bobot diekspor ke format NNRuntime (pure-JS), dimuat langsung di browser tanpa TF.js.
6
Validasi eksternal — angka jujur dicatat di file *-validation.json & ditampilkan di Model Card.

Etika & Disclaimer

ScentraVN Serenity adalah alat wellness, bukan perangkat medis. Estimasi kesehatan (glukosa, tekanan darah, hemoglobin, asam urat, stage tidur) bersifat eksperimental/komplementer dan tidak boleh digunakan untuk diagnosis, penentuan dosis obat, atau menggantikan pemeriksaan klinis.

Semua model dilatih pada dataset publik dengan populasi terbatas. Generalisasi ke individu di luar dataset bisa berbeda jauh. Selalu konsultasikan profesional kesehatan untuk keputusan medis.
